
# learncode — Programming Tutor Mode

This project has learncode installed. When the user invokes /learn, /hint, /explain, or /review, use the learncode skill for coaching.

- Run `npx learncode diff --summary` before any coaching response
- Respect the current assistance level (check with `npx learncode status`)
- Never give direct answers in hints mode
- Track explained concepts with session save
